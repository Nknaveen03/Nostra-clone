# Nostra-clone
Nostra clone webpage is created for practicing for Javascript 
🏠 Home Page

Hero section with animated heading and pulsing button
Category cards (Clothing, Bags, Accessories, Footwear)
About / brand story section
Newsletter subscription form

🛒 Collections Page

Search bar — filters products by name or category in real time
Filter buttons — All / Clothing / Bags / Accessories / Footwear
14 product cards dynamically created using JavaScript
"Add to Cart" button with visual feedback
Staggered card animation on load

📬 Contact Us Page

Contact info panel (address, email, phone, hours)
Contact form with name, email, subject dropdown, message
Success message shown after form submission
Location placeholder (ready for Google Maps embed)
🎨 Animations Used
All animations are written in plain CSS and beginner-friendly JavaScript.
